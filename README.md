# RESTful-API

### RESTful API Project  
*A lightweight RESTful API built using JavaScript, Express, and MongoDB with Mongoose. This project demonstrates CRUD operations, efficient data handling, and seamless data exchange using JSON. Developed and tested with Visual Studio Code and Insomnia.*  

### Token Authentication Project  
*A simple implementation of token-based authentication using JavaScript, Express, and MongoDB with Mongoose. The project leverages JSON Web Tokens (JWT) for secure user login and access control. Developed and tested with Visual Studio Code and [JWT.io](https://jwt.io/).*
